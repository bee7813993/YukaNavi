# ゆかりちゃん パーツ分けPSD

既存の `yukari_mascot_transparent.png` を元にした、動作用の初期PSD素材です。

## ファイル

- `yukari_parts_source.psd`: パーツ分けPSD。1024x1536 / RGBA
- `parts/*.png`: PSDと同じ各レイヤーを個別PNG化したもの。すべて1024x1536 / RGBA
- `manifest.json`: レイヤー順・非表示レイヤー・左右表記の説明
- `yukari_parts_grid_preview.png`: パーツ一覧プレビュー
- `yukari_parts_recomposed_preview.png`: 可視レイヤーを重ねた確認用プレビュー

## レイヤー方針

- 左右表記は、画面から見た左右です。
- 口閉じレイヤーと参照用の全身レイヤーは非表示にしています。
- 元イラストに含まれるため、指定粒度に加えて `legs_boots`、`headset_ribbon`、`tablet_extra` も補助レイヤーとして含めています。
- マイクとタブレットは独立レイヤーですが、腕側にも重なりを少し残しています。初期表示で穴が出にくく、必要に応じてリギング側でマスク調整しやすい作りです。

## 注意

これは既存の1枚絵から分解した初期リギング用PSDです。隠れている裏側の描き足しは完全な手描き再構成ではないため、大きく髪・腕・顔を動かす用途では追加の描き足し調整を推奨します。
