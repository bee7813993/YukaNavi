### 概要 (4000文字以内)

```
ゆかナビは、カラオケリクエストシステム「ゆかり」のためのモバイルコンパニオンアプリです。
パソコンで動いている「ゆかり」のサーバーにスマホから接続して、曲さがしから予約、順番の確認までを手もとでかんたんに行えます。

■ ご利用には「ゆかり」が必要です
本アプリは単体ではカラオケの再生はできません。カラオケ会の主催者などが運営する「ゆかり」サーバー (無料・オープンソース) に接続して使用します。
「ゆかり」について: https://github.com/bee7813993/KaraokeRequestorWeb

■ できること
・曲をさがす — キーワード検索のほか、歌手名・作品名からの検索に対応
・予約する — うたう人の名前とコメントを添えて、その場で予約
・順番を確認 — 予約一覧をリアルタイムに表示。自分の予約はひとめでわかる色分け
・再生の確認 — いま流れている曲や再生の状態を表示
・マイページ — うたった履歴、お気に入りの曲、あとでうたうリスト、お気に入り検索ワードを管理
・Google Drive バックアップ (任意) — マイページのデータを自分の Google アカウントに保存して、機種変更しても引き継ぎ
・QR コードでかんたん接続 — 部屋の QR コードを読み取るだけで接続設定が完了
・きせかえ — スキンを切り替えて、自分だけの見た目に

■ こんな場面で
・友人どうしのカラオケオフ会
・イベント・コミュニティのカラオケ企画
・自宅でのカラオケパーティー

■ プライバシー
広告 SDK やアクセス解析ツールは入っていません。詳しくはプライバシーポリシーをご覧ください。
https://ykr.moe/apps/yukanavi/privacy.html
```

### データセーフティ申告の下書き

フォームの選択肢に合わせた下書き。最終判断は申告画面の説明文を見ながら行うこと。

| 質問 | 回答 | 根拠 |
|------|------|------|
| データを収集または共有するか | 収集する (共有はしない) | 下記参照 |
| 送信の暗号化 | 「ユーザー設定のサーバー」宛は HTTP の場合あり / Google・中継サーバー宛は HTTPS | 部屋のゆかりサーバーは LAN 内 HTTP が一般的 |
| 削除リクエスト手段の提供 | あり | プライバシーポリシー「データの削除」参照 |

収集データの内訳:

- **個人情報 > 名前**: 収集する / 任意 / アプリの機能 / 共有なし
  — 「うたう人」の表示名を接続先ゆかりサーバーに送信 (予約機能)
- **個人情報 > メールアドレス**: 収集する / 任意 / アカウント管理 / 共有なし
  — Google ログイン利用時のみ。表示・識別に使用
- **アプリのアクティビティ > その他の操作**: 収集する / 任意 / アプリの機能 / 共有なし
  — 予約内容 (曲・コメント) と匿名 ID (UUID) を接続先サーバーに送信
- 位置情報・連絡先・広告 ID・診断情報: 収集しない

注: 「収集」の大半は開発者ではなく **利用者が接続先に選んだゆかりサーバー** への送信。
マイページの Google Drive バックアップは利用者自身のアカウントのアプリ専用領域への
保存で、開発者はアクセスできない。申告フォームの補足欄・ポリシーで明記する。

## App Store「アプリのプライバシー」申告の下書き

- **収集するデータ**:
  - 名前 (アプリの機能 / ユーザーに関連付けられる / トラッキングなし)
  - メールアドレス (アプリの機能 / ユーザーに関連付けられる / トラッキングなし) — Google ログイン任意
  - ユーザー ID (アプリの機能 / ユーザーに関連付けられる / トラッキングなし) — アプリ生成の匿名 UUID
- **トラッキング**: なし (広告・解析 SDK 不使用)
