ゆかナビ

カラオケリクエストツール「ゆかり」を、手元のスマートフォンからもっと使いやすく。

Android版 公開準備中

ABOUT THE APP

歌いたい曲へ、迷わずたどり着く。

ゆかナビは、カラオケリクエストツール「ゆかり（KaraokeRequestorWeb）」に接続して、曲の検索やリクエスト、予約状況の確認を行うための公式アプリです。

FEATURES

カラオケの時間に必要な機能を、ひとつに。

01

曲を探してリクエスト

さまざまな検索方法から曲を探し、接続中の「ゆかり」へリクエストできます。

02

QRコードでかんたん接続

カラオケルームに表示されたQRコードを読み取り、接続先をすばやく設定できます。

03

履歴とお気に入りを手元に

履歴やお気に入りを端末に保存し、任意でGoogle Driveと同期できます。
