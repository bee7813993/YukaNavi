SUPPORT

ゆかナビ サポート

接続や使い方で困ったときの確認事項と、お問い合わせ先をご案内します。

お問い合わせ前の確認

「ゆかり」に接続できません

スマートフォンと「ゆかり」を動かしている端末が、同じローカルネットワークに接続されているか確認してください。接続先URLは手入力のほか、表示されたQRコードから設定できます。

QRコードを読み取れません

端末の設定で、ゆかナビにカメラの利用が許可されているか確認してください。カメラを利用できない場合も、接続先URLを手入力できます。

QRコードが画面から遠い場合は、ズームスライダーやピンチ操作で拡大できます。それでも読み取れないときは「写真で読み取る」で、カメラアプリのズームを使って撮影した写真から読み取れます。

Google Driveと同期できません

インターネット接続とGoogleアカウントのログイン状態を確認してください。認証が切れている場合は、一度ログアウトしてから再度ログインしてください。

CONTACT

お問い合わせ

不具合の報告やご質問は、GitHubのIssueから受け付けています。端末の機種、OSのバージョン、発生した操作、表示されたメッセージを添えていただくと確認がスムーズです。

GitHub Issuesを開く

パスワード、認証コード、個人を特定できる情報は投稿しないでください。公開したくない情報を含む場合は、投稿前に該当部分を伏せてください。
