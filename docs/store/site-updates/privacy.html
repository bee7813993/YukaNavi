PRIVACY POLICY

プライバシーポリシー

YFRTeamが提供するアプリ「ゆかナビ」における、利用者情報の取り扱いについて定めます。

制定日：2026年7月16日（2026年7月19日 改定）

1. 適用範囲

本ポリシーは、YFRTeam（以下「当方」）が提供する「ゆかナビ」（以下「本アプリ」）に適用されます。本アプリから接続するカラオケリクエストサーバーや外部サービスについては、それぞれの運営者が定める規約やポリシーも適用される場合があります。

2. 本アプリが取り扱う情報

接続設定および認証情報

本アプリは、利用者が指定したカラオケリクエストサーバーのURL、認証設定、アプリ内で生成または連携されたユーザー識別子などを端末内に保存します。

アプリの利用に伴う情報

曲の検索条件、リクエスト内容（うたう人として入力した名前、コメント、選曲した曲の情報など）、曲情報の修正内容、予約状況、履歴、お気に入り、表示設定などを取り扱います。接続先サーバーの機能を利用するために必要な情報（リクエスト内容や利用者識別子など）は、そのサーバーへ送信され、予約情報や履歴として保存されます。また、通信にともない、接続先サーバーにはIPアドレスや端末のユーザーエージェントが記録される場合があります。

カメラ

接続先を設定するQRコードの読み取りにカメラを使用します。カメラ映像および「写真で読み取る」機能で撮影した写真は、端末内でQRコードを解析するためにのみ使用します。撮影した写真はアプリ専用の一時領域にのみ置かれ、端末のフォトライブラリへの保存や外部への送信は行いません。読み取った接続先URLは、利用者の確認および操作に基づいて端末内へ保存されます。

利用者が選択したファイル

きせかえ機能では、利用者が選択した画像、動画、音声、スキンファイルを読み込み、端末内に保存します。利用者が共有操作を行った場合を除き、これらのファイルを当方のサーバーへ送信することはありません。

GoogleアカウントおよびGoogle Drive

利用者が任意でGoogle連携を有効にした場合、Googleアカウントの識別情報およびメールアドレス、認証トークンを取り扱います。履歴、お気に入り、保存した検索条件などのマイページデータは、利用者自身のGoogle Driveのアプリデータ領域に保存・同期されます。

3. 利用目的

  • 本アプリの機能を提供するため
  • 利用者が指定したカラオケリクエストサーバーへ接続するため
  • 設定、履歴、お気に入りなどを保存・同期するため
  • 不具合への対応およびサービスの安全な運営のため

4. 外部サービスへの送信

本アプリは、機能の提供に必要な範囲で、次の送信先と通信します。

  • 利用者が指定したKaraokeRequestorWebサーバー
  • Google OAuth認証およびGoogle Drive API
  • Google認証を本アプリへ引き渡すための中継サーバー
  • きせかえの追加コンテンツを取得するための当方の配信サーバー（利用者が入力した追加コードを送信します）

Google認証の中継サーバーでは、認証情報をアプリへ引き渡すために一時的に保持し、受け渡し完了時または有効期限（最長10分）の経過後に削除します。中継サーバーが認証情報を継続的に保存することはありません。

5. 通信の安全性

Google関連サービスおよび当方が提供するGoogle認証中継サーバーとの通信にはHTTPSを使用します。

本アプリの性質上、利用者が指定する同一LAN内のカラオケリクエストサーバーへHTTPで接続する場合があります。HTTP通信は暗号化されません。信頼できるローカルネットワーク内で利用し、インターネット上の不明な接続先を指定しないでください。

6. 広告およびアクセス解析

本アプリは、第三者の広告配信サービスおよび利用者を追跡するアクセス解析サービスを使用していません。

7. Googleユーザーデータ

本アプリによるGoogle APIから取得した情報の使用および他のアプリへの転送は、限定的使用（Limited Use）の要件を含むGoogle API Services User Data Policyに準拠します。

8. 保存期間と削除

端末内の情報は、利用者が設定やアプリデータを削除するか、本アプリをアンインストールするまで保存されます。Google Driveへ同期した情報は、利用者のGoogleアカウントの管理下で保存されます。

Google連携のデータは、アプリ内の「マイページ > 連携 > 連携データを削除」から、Google Drive上のバックアップファイルの削除とアクセス権の取り消し(ログアウトを含む)をいつでも行えます。このほか、本アプリからログアウトすることや、Googleアカウントの接続済みアプリの管理画面からアクセス権を取り消すこともできます。接続先サーバーに保存された情報の削除については、そのサーバーの管理者へお問い合わせください。

9. 本ポリシーの変更

機能の追加、法令または外部サービスの変更などに応じて、本ポリシーを改定することがあります。重要な変更がある場合は、本ページまたはアプリ内でお知らせします。

10. お問い合わせ

本ポリシーおよび本アプリに関するお問い合わせは、サポートページからご連絡ください。
